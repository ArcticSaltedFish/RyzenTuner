2. 前台运行 IDE
3. 室温 26 ℃
4. 测试 CPU：AMD Ryzen™ 7 PRO 6850HS

    * 基础频率：3.2GHz
    * 最大加速时钟频率：4.7GHz
    * 最高温度：95°C
